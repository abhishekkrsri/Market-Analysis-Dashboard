# 📈 Market Analysis Dashboard

![Python](https://img.shields.io/badge/Python-3.11-blue)
![Streamlit](https://img.shields.io/badge/Streamlit-Dashboard-red)
![Plotly](https://img.shields.io/badge/Plotly-Visualization-green)
![MetaTrader5](https://img.shields.io/badge/MetaTrader5-Trading-orange)

An interactive financial market analysis dashboard built using **Python**, **Streamlit**, **MetaTrader 5**, **Plotly**, and **Pandas**.

The application enables traders and analysts to fetch historical market data, perform statistical analysis, identify breakout patterns, analyze recurring market behavior across different time windows, visualize insights through interactive charts, and export the analysis as CSV reports.

This project was developed as part of a **Software Engineer / Data Analyst Internship Take-Home Assignment**.

---

# 🚀 Features

- 📊 Fetch historical market data directly from MetaTrader 5
- 🌍 Multiple Trading Symbols
- ⏱ Multiple Timeframes (M1, M5, M15, H1, H4, D1)
- 📅 Historical Analysis Period Selection
- 📈 Interactive Candlestick Chart
- 📊 Key Performance Indicators (KPIs)
- 📈 Market Insights
- 📉 Candle Statistics
- 📊 Return Statistics
- 🚀 Breakout Analysis
- ❌ False Breakout Analysis
- 🕒 Time Window Analysis
- 🏆 Best Trading Time Windows
- 📌 Maximum Favorable Excursion (MFE)
- 📌 Maximum Adverse Excursion (MAE)
- 📊 Return Distribution Histogram
- 📈 MFE vs MAE Scatter Plot
- 📋 Historical Market Data Table
- 📥 Export Analysis as CSV

# ⚙ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/abhishekkrsri/market-analysis-dashboard.git

cd market-analysis-dashboard
```

## 2. Install Dependencies

```bash
pip install -r requirements.txt
```

## 3. Install MetaTrader 5

- Install MetaTrader 5
- Log in to your trading account
- Keep MetaTrader 5 running before starting the dashboard

---

# ▶ Running the Project

Run the Streamlit dashboard

```bash
streamlit run dashboard/app.py
```

Run the analysis script

```bash
python main.py
```
